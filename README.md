# INSTALLATION

Add the following line to your ~/.bashrc file:

```
export PYTHONPATH=$PYTHONPATH:/path/to/where/this/library/is # (the root folder, NOT the kaleido folder)
```

followed by:

```
$ source ~/.bashrc
```

in a terminal, to refresh your environment variables.
